## Description
Recruiment challenge in [Nest](https://github.com/nestjs/nest)

## Installation
------------

Download with `git`:

``` bash
$ git clone https://github.com/josemanuelcarretero/bnext-recruitment-challenges.git
```

Install dependencies with `yarn`:

``` bash
$ npm install -g @nestjs/cli
$ yarn install
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
